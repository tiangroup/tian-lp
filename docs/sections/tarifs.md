## Поля item

| название    | суть                                 |
| ----------- | ------------------------------------ |
| title       | название                             |
| target      | панч-лайн (например, для кого тариф) |
| description | описание в html                      |
| price       | цена                                 |
| selected    | bool; отмеченный тариф               |
